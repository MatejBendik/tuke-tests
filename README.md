## How to use:

```bash
  git clone https://github.com/MatejBendik/tuke_tests
```

Go to the project directory

```bash
  cd tuke_tests
```

Go to the specific subject folder e.g. prg

```bash
  cd prg
```

Go to the specific problem set e.g. ps2

```bash
  cd ps2
```

Paste your solution e.g. k.c

```bash
  cp ~/path/to/k.c k.c
```

Start tests

```bash
  make test
```

Clean .o files

```bash
  make clean
```

## To update tests:

```bash
  git pull
```
